About Speech Recognaization and Language Detection

Overview

The Speech Translator and Transliterator application is designed to help users convert spoken language into written text, translate it into another language, or transliterate it into a different script. This tool is particularly useful for language learners, travelers, and anyone who needs to communicate across language barriers.

Features

  • 🗣️Speech Transcription: Convert spoken words into written text in multiple languages.
  • 🌐Text Translation: Translate transcribed text into a different language.
  • 🔤Text Transliteration: Convert text from one script to another, making it readable in a different writing system.
  • 🔊Speach-To-Text: Listen to the translated or transliterated text using a text-to-speech engine.

How It Works

The application operates in several steps:

  1. Select Language: The user chooses the language they will speak in.
  2. Transcribe Speech: The application listens to the user and converts their speech to text.
  3. Choose Action: The user decides whether to translate or transliterate the transcribed text.
  4. Perform Action:
    • If translating, the user selects the target language, and the text is translated.
    • If transliterating, the user selects the target script, and the text is converted to that script.
  5. Text-to-Speech: The application can read the translated or transliterated text aloud.

Benefits

  • 📚Language Learning: Improve language skills by listening and seeing translations and transliterations.
  • ✈️Travel: Communicate more effectively when traveling to different countries.
  • ♿Accessibility: Assist those with speech or reading difficulties by providing spoken and written translations.

Getting Started

To use this application, follow these steps:

  1. Install the required Python libraries: unidecode, speech_recognition, googletrans, indic_transliteration, pyttsx3.
  2. Run the main script.
  3. Follow the prompts to select languages, speak, and choose actions.

Requirements

Ensure you have the following libraries installed:

  • unidecode
  • speech_recognition
  • googletrans
  • indic_transliteration
  • pyttsx3
