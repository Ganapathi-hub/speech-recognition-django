# Speech Recognition, Translation and Transliteration System

## 📌 Project Description

This project is a Django-based web application that performs **Speech-to-Text conversion**, **Language Translation**, and **Transliteration** using Python libraries.

Users can speak through a microphone, and the system converts speech into text and provides translated or transliterated output.

---

## 🚀 Features

* Speech to Text Conversion
* Multi-language Translation
* Transliteration Support
* User Authentication (Login / Signup / OTP verification)
* Feedback System

---

## 🛠 Technologies Used

* Python
* Django
* SpeechRecognition
* Googletrans
* SQLite3
* HTML, CSS, JavaScript

---

## ⚙️ Installation

Clone the repository

git clone https://github.com/Ganapathi-hub/speech-recognition-django.git

Go to project folder

cd speech-recognition-django

Install dependencies

pip install -r requirements.txt

Run migrations

python manage.py migrate

Run server

python manage.py runserver
